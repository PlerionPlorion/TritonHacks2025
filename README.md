# Vite + Svelte + Electron + Tailwind Base

Boilerplate using **Vite**, **Svelte**, **Electron**, and **Tailwind CSS**.

---

## Installation


1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Build the app:

   ```bash
   npm run make
   ```
